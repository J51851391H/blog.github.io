Helpful Resources

Here are a few of my favorite resources to learn Korean.

Learn to Type in Korean:

  • TaDak TaDak: Great web app to learn the basics.
  • Fast Fingers (Korean Version): Fun web app to practice getting faster. The english version is pretty fun too.

Help with Pronunciation:

  • Motivate Korean's Perfect Your Korean Pronunciation ($75): The best course I've found on korean pronunciation.
  • Sweet and Tasty TV's Korean Work of the Week Series: A fun, free way to learn korean pronunciation.

Courses:

  • Pimsleur's Learn to Speak and Understand Korean: This is an audio only course available through Audible. The style of Korean taught is a bit stiff, but it's an excellent way to practice while you're on the go...especially if you're a beginner.
  • Talk To Me In Korean: This is my favorite resource. TTMIK has podcasts, online courses, books, videos...it's simply fantastic. Plus, they have loads of free material.

Tools:

  • Anki: Anki is an open source flash card program available for PC, Mac, Linux, Android, and iOS.
  • iTalki: Inexpensive, online tutors. I've had HUGE success with meeting regularly with a teacher.

Dictionaries:

  • KoreanClass101.com's Dictionary
  • Learn With Oliver's Dictionary
  • Naver's Papago Dictionary
